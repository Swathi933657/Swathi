What is a Scaler in Python (Machine Learning)?
ans:A scaler is a tool used to change the range of data so that all values are on a similar scale.
Why?
Because ML algorithms work better when features are in similar ranges.
Example:
Suppose your data has:
Age ‚Üí 18 to 60
Salary ‚Üí 20,000 to 100,000
üëâ These values have different scales, which may confuse the ML model.
So we scale them to a similar range like 0 to 1.

Common Scalers in Python (sklearn):
| Scaler Name        | Purpose                                    |
| ------------------ | ------------------------------------------ |
| **StandardScaler** | Converts data to have mean = 0 and std = 1 |
| **MinMaxScaler**   | Scales data between 0 and 1                |
| **RobustScaler**   | Used when outliers are present             |
| **Normalizer**     | Converts values to unit vector length      |

eg code:
from sklearn.preprocessing import StandardScaler
import numpy as np

# Sample data
data = np.array([[10], [20], [30], [40], [50]])

# Create scaler object
scaler = StandardScaler()

# Fit and transform data
scaled_data = scaler.fit_transform(data)

print(scaled_data)


waht is mean median and mode?
ans:These are used in statistics to find the central value in a dataset.
1)Mean (Average)
The mean is the sum of all numbers divided by total count.
Formula:

Mean=Total number of values /Sum of all values
Example:
Numbers: 5, 10, 15, 20
Mean = (5 + 10 + 15 + 20) / 4 = 12.5

2Ô∏è‚É£ Median (Middle Value)
The median is the middle value when numbers are sorted.
Steps:
Arrange numbers in ascending order
If odd count ‚Üí middle one
If even count ‚Üí average of two middle values
Example:
Numbers: 5, 10, 15, 20, 25 ‚Üí Median = 15
Numbers: 5, 10, 15, 20 ‚Üí Median = (10 + 15) / 2 = 12.5

3Ô∏è‚É£ Mode (Most Frequent Value)
The mode is the number that appears the most times.

Example:Numbers: 2, 4, 4, 6, 8, 4 ‚Üí Mode = 4
There can be:
One mode ‚Üí Unimodal
Two modes ‚Üí Bimodal
More than two ‚Üí Multimodal
No mode ‚Üí If all values are different
