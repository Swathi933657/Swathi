3.waht is ravel and stack?
ans:ravel() converts a multi-dimensional array into a 1D array.
use ravel():
It is faster than flatten()
It does not create a copy, it gives a view (original array changes too)
Used when you want to loop or access linear positions

flatten() vs ravel() (easy difference):
| Function    | Returns  | Copy or View?                 |
| ----------- | -------- | ----------------------------- |
| `ravel()`   | 1D array | **View** (linked to original) |
| `flatten()` | 1D array | **Copy** (separate array)     |


stack:

âœ” np.dstack() â†’ join depth-wise (3D stack)
âœ” np.stack() â†’ create a new axis and join
